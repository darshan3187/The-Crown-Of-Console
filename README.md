# The Crown Of Console 👑

A premium, high-performance gaming cafe website for **The Crown Of Console**, Ahmedabad.

## 🚀 Tech Stack
- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the local development server:
```bash
npm run dev
```

### Production Build
Build the project for production:
```bash
npm run build
```
